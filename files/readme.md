### file system
The file system is designed in following way:

<pre>
/files
|— slides
|   |— slide_name.svs
|
|— patches
|   |— slide_id
|      |— patch001.png
|      |— patch002.png
|      |— …
|
|— annotations
|   |— slide_id/
|       |— updated_at_time001.annotations
|       |— updated_at_time002.annotations
|       |— …
|       |— initial_annotation.annotations
</pre>
